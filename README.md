# CUDA Programming Guide 中文版

> NVIDIA CUDA Programming Guide 官方文档中文翻译版

🌐 **在线文档**：https://bearneck.github.io/cuda-programming-guide-zh/

📖 **原文地址**：https://docs.nvidia.com/cuda/cuda-programming-guide/

## 简介

本仓库是 NVIDIA CUDA Programming Guide 的中文翻译版本，覆盖全部章节：

- 第一部分：CUDA 简介（编程模型、CUDA 平台）
- 第二部分：CUDA GPU 编程（CUDA C++、SIMT 内核、异步执行、内存管理）
- 第三部分：高级 CUDA（高级 API、多 GPU、驱动层）
- 第四部分：CUDA 高级特性（Graphs、协作组、动态并行等 20 个专题）
- 技术附录（计算能力、环境变量、语言扩展）
